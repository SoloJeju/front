## ⚙️기술 스택
- **언어** : React + TypeScript
- **빌드 툴** : Vite
- **스타일링** : TailwindCSS
- **라우팅** :  React Router
- **상태관리** : Zustand
- **데이터 캐싱** : Tanstack Query

## 🎀브랜치명 정의
- 영어로 통일, 케밥케이스로 작성
- 작성 형식: 브랜치 유형/브랜치명
- 작성 예시
  - feat/login-page

## 👑네이밍 컨벤션
- `camelCase` : 변수, 일반함수
    - ex) isError, handleSubmit
- `UPPER_CASE_SNAKE_CASE` : 상수
    - ex) BASE_URL
- `PascalCase` : 클래스, 컴포넌트 함수, 타입, 인터페이스, 컴포넌트 파일
    - ex) CardListProps
- `kebab-case` : 파일, 폴더
    - ex) main-layout, login-page
    - 컴포넌트 파일 및 폴더만 예외로 파스칼 케이스로 작성
